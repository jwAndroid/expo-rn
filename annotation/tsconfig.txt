리액트(ts) 셋업하면 자동으로 tsconfig 셋업해주는데
이 셋업을 리액트네이티브에 그대로 옮겨가져가서 사용하는게 맞을수도 있다고함

- 그리고 참고하자면
명령어는 패키지.js 에 있음

{
  "extends": "expo/tsconfig.base",
  "compilerOptions": {
    "target": "es6",
    "strict": true,
    "forceConsistentCasingInFileNames": true,
    "noFallthroughCasesInSwitch": true,
    "isolatedModules": true,
    "noEmit": true
  }
}
