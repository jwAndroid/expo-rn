우리회사 공유 깃허브주소에서
url 복사후
터미널 -> git clone url 후
폴더에 완료되었으면
vs코드 들어가서 폴더열고

컨트롤+` 로 vs터미널 열어주고
npm i 로 종속성 다운로드 업데이트

npm start or run start (처음배울때는 런 스타트가 더 나음)
해주면 우리서버 http://localhost:3000/
터미널 위  Local:            http://localhost:3000
  On Your Network:  http://192.168.0.6:3000
이런식으로 뜸

리액트 실행완료

패키지 -> src / App / 에서 주소/파라미터("/room/:id") (rest api 사용과 비슷) 넣게되면 특정한게 동작함.
주소창 치는건 전부다 @GET임

<Redirect>는 예외처리같은거 아무거나 막 치면 초기로 돌아감
