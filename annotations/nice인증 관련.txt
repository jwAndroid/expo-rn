위트립 nice 쪽가면 코드나올텐데
데이터를 가져올때 결과부터 설명하자면
const result = document.querySelector('pre').innerHTML;
ReactNativeWebView.postMessage(result);

HTML pre 태그안값을 가져와서 WEBVIEW에 띄워준다.
값을 다시 JSON 으로 파싱하여 가공된 데이터를 우리가 사용하는것이다.

NICE인증 관련 프로세스는
APP --> 백엔드요청후 URL 받기
APP이 URL을 받았다면 해당주소를 (EncodeData , m 데이타) 조합하여 나이스 웹뷰띄우기
웹뷰를 띄우고 사용자정보를 서버를 한번거친뒤 나이스에서 app으로 주는것
